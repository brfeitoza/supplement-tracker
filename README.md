# Rastreador de Suplementos Alimentares na Blockchain

Este é um contrato inteligente Solidity para rastreamento e autenticação de suplementos alimentares na blockchain Ethereum. O contrato foi desenvolvido e testado usando o ambiente Remix.

## Funcionalidades Principais

- Adição de informações sobre suplementos alimentares, incluindo nome, fabricante, conteúdo de proteínas, carboidratos, gorduras e data de validade.
- Assinatura digital das informações pelos proprietários autorizados.
- Verificação de autenticidade das informações usando assinaturas digitais.

## Como Usar

1. Abra o ambiente Remix (https://remix.ethereum.org/).

2. Crie um novo contrato ou importe o código-fonte do contrato inteligente fornecido neste repositório.

3. Compile o contrato inteligente no ambiente Remix.

4. Implante o contrato inteligente em uma rede Ethereum de sua escolha (por exemplo, rede local, Ropsten, etc.).

5. Interaja com o contrato inteligente usando as funções fornecidas, como `addSupplement`, `signSupplement`, `verifySupplementAuthenticity`, etc.

6. Use o ambiente Remix para monitorar os eventos emitidos pelo contrato para acompanhar as atividades.

7. Consulte a documentação interna do contrato para obter informações detalhadas sobre cada função e variável.
